How to Deploy on GitHub Pages

01
Create a GitHub Repository
Go to github.com → New repository. Name it something like upsc-revision. Keep it public.
02
Upload All HTML Files
Upload all 8 files: index.html, history.html, geography.html, polity.html, economy.html, environment.html, science.html, art-culture.html to the root of your repo.
03
Enable GitHub Pages
Go to your repo → Settings → Pages → Source: Deploy from branch → Branch: main → folder: / (root) → Save.
04
Access Your Site
Your site will be live at https://YOUR-USERNAME.github.io/upsc-revision/ within a few minutes. Share this URL for revision!
